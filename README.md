# Memory Game

### Features
- User can select cards theme between number or icons
- User can select game grid size
- Game allows to play up to 4 players at the same time
- Game keep tracks of users moves and time

### Tech stack
- HTML 5
- CSS 3
- SCSS
- Javascript ES6
- Typescript
- React
- React Context API
